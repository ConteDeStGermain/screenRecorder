This is a simple screen Recorder designed to work on windows for google Classroom

Gettting started: 
Ensure node is installed.

change example.env.txt to .env and put your credentials there.

in your powershell terminal, navitage to this projects's directory do:
- npm install

In the project directory do the following to start the script:
- node FileNameOfTheDay.js

for the python requirements, check requirements.txt

Ensure that the start time corresponds with your timzone. Change line 10.
